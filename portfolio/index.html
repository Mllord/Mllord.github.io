Deep Web // Z-Axis Interactive Website

An interactive website that explores the layers of the internet from a vertical perspective. Scrolling reveals different depths of network infrastructure, from surface web content down to server protocols. Built with custom CSS scroll-snap and JavaScript intersection observers to create a seamless Z-axis storytelling experience. The site visualizes how data travels through physical hardware, code, and user interfaces.

Fluid Simulation // Touch Designer

A real-time hand tracking fluid simulator built in TouchDesigner. The system responds to hand gestures and audio input, creating evolving, mirrored fluid patterns that shift with sound intensity. The interface remains minimalist and reflective, allowing the flowing simulations to take center stage. Gesture recognition enables unique interactions — users can direct, split, and merge fluid streams with simple hand movements. The project explores synesthetic perception, where the auditory and visual senses merge into a seamless feedback loop.

Bio

I’m an Australian-Japanese designer focused on creating immersive game worlds and characters. I work across both character and environment design, using tools such as Blender, Maya, Unity, and Unreal Engine to build detailed, interactive assets that bring digital spaces to life.

My practice is driven by world-building, crafting environments and characters that feel cohesive, atmospheric, and grounded within a larger narrative. From stylised assets to more realistic forms, I aim to balance strong visual design with technical execution, ensuring my work translates effectively into real-time engines.

I approach every project with a results-driven mindset, combining creativity with discipline to see ideas through from concept to completion. Constantly refining my skills and exploring new techniques, I’m focused on pushing the quality of my work while contributing meaningfully to game development teams.
