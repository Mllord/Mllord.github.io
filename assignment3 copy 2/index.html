The Internet

A massive global network linking billions of devices. It enables digital communication, information sharing, and access to services worldwide through standardized data transmission.

Web Services

Systems built on the internet like email, cloud storage, streaming, and messaging platforms. These provide interactive experiences and enable digital productivity, entertainment, and communication.

Websites & Applications

Individual tools or destinations online—like YouTube, Wikipedia, or banking apps. Each serves specific functions and caters to user needs like searching, shopping, watching, or managing tasks.

Pages & Interfaces

The screens users interact with—homepages, dashboards, profiles, etc. These visual layouts organize content, navigation, and interactive elements into a user-friendly experience.

Content & Funtionality

Text, images, videos, links, buttons, and forms. These are the visible parts users read, watch, click, and fill out to navigate and use a site or app.

Codes & Scripts

Behind-the-scenes instructions written in languages like HTML, CSS, and JavaScript. This code defines how everything looks and behaves, from page layout to animations and responses to clicks.

Protocols & Data

Rules and structures (like HTTP, TCP/IP, JSON) that govern how devices exchange information. They ensure data moves accurately and securely across the network.

Servers & Infrastructure

The physical and digital backbone: computers, data centers, cables, and routers. These machines host websites, process requests, and deliver data across the globe.
